# takahashi

有了趁手的工具以后，一些以前做起来很麻烦很费事的东西，现在都变得无比简单。我算手慢的，撸这个高桥流投影片模版，也就是晚饭后睡觉前这会儿功夫。如果不知道高桥流是什么，可以看这里。

操作方式很简单。桌面浏览器按键盘的右（➡️）、下（⬇️）和空格键向后翻页，按左（⬅️）和上（⬆️）向前翻页。移动设备上左划向后翻页，右划向前翻页。
