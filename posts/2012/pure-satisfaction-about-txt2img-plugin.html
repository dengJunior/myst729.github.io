# 单纯的满足感 — 关于 txt2img 插件

txt2img 山寨长微博是我开发的一个 WordPress 插件。它可以将文章的文字内容转换成图片，分享到新浪微博。借助这种手段，我们可以突破新浪微博 140 个字的限制，发布篇幅较长的微博内容。

最初开发这个插件仅仅是因为自己挖了个坑跳进去。坑填完之后自然免不了吼两声。过后就是有人建议我发布到 WordPress 官方插件目录，我觉得也不错，就提交了。因为 WordPress 插件目录的代码是用 SVN 管理的，觉得不爽，所以又放了一份到 GitHub。

从 WordPress 官方提供的数据来看，九个月才不温不火的 400 多次下载，这个插件也就那样了吧。然而由于插件本身使用场景所限，决定了用户都是能写，并且愿意写长文的人。在这个发微博易如反掌的时代，自建博客写长文章是非常难的一件事情。不管是 CEO 用户，还是各位普通站长和程序员，以及其他辛勤劳作的朋友，坚持写长文章都是很不容易的。所以尽管自己的插件不够火，但用户质量还是挺高的，这样一想就觉得很满意，满足。插件从今年 2 月份的第一个版本发布到现在，质量越来越稳定，功能也逐渐增强，这都得感谢各位高质量用户的积极反馈和帮助。甚至有朋友直接给我发 patch，真是太感人了。

当然了，插件仍然有一些不足，也不可能做到让所有人都满意。比如黄一孟就不喜欢现在用的 Droid Sans Fallback 字体，对此我在微博上给予了回复。我会尽量满足用户的需求，前提是不违背我的原则。

开源是让人愉快的事，就是因为这种单纯的满足感。
