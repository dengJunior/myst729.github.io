# 博客迁移到 Hexo

本来说要改 blog 主题，太懒一直拖着不想动手。空间商又说要涨价，心想本来一年就写不了多少东西，还是别花这钱了，加上对积重难返的 WordPress 兴趣大不如前，干脆往 GitHub 搬吧……

搜能跑在 NodeJS 上跟 Jekyll 类似的工具，发现有个台湾人写的 Hexo 还不错，安装和部署都很容易，生成文章速度快更是一大亮点。花了大概一周的时间，边看文档边东拼西凑抄出一个主题来。准备过后把旧的文章整理一下，慢慢都搬过来。forcefront.com 这个域名也不打算再用了，反正我也不炒域名，无所谓。
